# GTA:SA Discord Rich Presence (Tradução PT-BR)

Este é um fork de tradução do projeto [gtasa-discord-richer-presence](https://github.com/MikMik1011/gtasa-discord-richer-presence) de MikMik1011.

**Todo o texto de status visível no Discord foi traduzido para o Português Brasileiro (PT-BR).**

---

## O que é isso?

Este é um plugin `.asi` para o Grand Theft Auto: San Andreas que exibe um status detalhado da sua atividade no jogo (Rich Presence) diretamente no seu perfil do Discord.

## Funções do Mod

O status no Discord é dinâmico e muda com base no que você está fazendo no jogo:

* **A Pé:** Mostra seu dinheiro e porcentagem de vida.
* **Em um Veículo:** Mostra o nome do veículo atual e a estação de rádio que você está ouvindo.
* **Nível de Procurado:** Mostra seu nível de procurado e a arma que está usando no momento.
* **Em Missão:** Mostra o nome da missão que você está jogando.
* **Imagens Interativas:** Ao passar o mouse sobre as imagens (grande e pequena) no status, você verá a localização atual e o horário dentro do jogo.
* **Suporte ao SAMP:** Ao jogar no SA-MP, seu status mostrará seu nickname e o nome do servidor.

## Instalação

1.  Você precisa de um **ASI Loader** para executar este mod. Se você não tiver um, [este é recomendado](https://github.com/ThirteenAG/Ultimate-ASI-Loader/releases).
2.  Baixe o arquivo `RichDiscord-MrGeH.asi` da nossa [página de Releases](https://github.com/MrG3H/Rich-Discord-GTA-San-Andreas/releases/tag/1.0)
3.  Coloque o arquivo `RichDiscord-MrGeH.asi` diretamente no diretório principal do seu GTA: San Andreas (onde fica o `gta-sa.exe`).
